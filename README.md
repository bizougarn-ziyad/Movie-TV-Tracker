# Movie / TV Show Tracker – Front-End

## Description
Movie / TV Show Tracker est une application web permettant aux utilisateurs de découvrir, rechercher et organiser les films et séries qu’ils ont regardés ou souhaitent regarder.

Ce dépôt contient **uniquement la partie Front-End** du projet.

## Landing Page
La landing page est la première page visible lors de l’accès au site, sans authentification.

### Fonctionnalités :
- Affichage des **films et séries populaires**
- Sections :
  - Popular Movies
  - Popular TV Shows
- Cartes visuelles (affiche, titre, note)
- Boutons :
  - **Login**
  - **Sign Up**

---

## Fonctionnalités Front-End
- Landing Page (contenus populaires)
- Authentification (interfaces)
- Recherche de films et séries
- Fiches détaillées
- Notation et commentaires (UI)
- Listes personnalisées
- Historique et statistiques
- Tendances
